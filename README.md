# Excel Split & Merge (с сохранением форматирования)

Небольшое веб‑приложение на Streamlit для:

- Разделения большого Excel-файла на части  
- Объединения нескольких Excel-файлов обратно в один  
- С сохранением форматирования (цвета, шрифты, границы, ширина колонок)

---

## Зачем это нужно

Когда отчёт слишком большой, его неудобно:
- отправлять по почте,
- загружать в систему,
- обрабатывать вручную.

Я сделал это приложение, чтобы:
- разбивать Excel на равные части,
- сохранять внешний вид таблиц,
- потом объединять части обратно без потери форматирования.

Важно:  
Используется `openpyxl`, а не `pandas`, потому что pandas удаляет всё визуальное оформление.

---

## Возможности

### Разделение Excel
- Загружается `.xlsx` файл
- Первая строка считается заголовком
- Можно выбрать размер части (по количеству строк)
- Каждая часть сохраняет:
  - цвета ячеек
  - шрифты
  - границы
  - объединённые ячейки
  - ширину колонок

---

### Объединение Excel через ZIP

- Загружаются ZIP-архивы
- В каждом архиве ожидается 1 Excel-файл
- Все файлы объединяются друг под другом
- Форматирование сохраняется
- Дубликаты не удаляются (объединение «как есть»)
